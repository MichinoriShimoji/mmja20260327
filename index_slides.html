伊良部ことばの mmja

伊良部のヤギども・・・酷すぎる

伊良部島は宮古島に次いで大きな島として、宮古島と兄弟のように語られる存在である。世の常として、互いの島および住人への対抗心が芽生えるもので、伊良部の人間は宮古の人間（特に平良の人間）を指して pžžara-pžguru「平良の冷たいやつら」といい、宮古の人間は伊良部の人間を指して irav-pinza「伊良部のヤギども」という。伊良部の人間は mmja（「もう」を表す副詞、談話標識）を発話中に多用することから、ヤギが鳴いているように聞こえるという揶揄である。実際に伊良部島方言の談話には mmja が多用され、伊良部出身の人を見抜く手掛かりになるほどである。

下地理則（近刊）「宮古語伊良部島方言」長屋尚典（編）『図説 世界の言語 — 危機言語と言語の多様性』朝倉書店 より引用。

川田他（2009）の主張
池間方言の mmya は「（従属節が描く）背景となる情報」を標示する (p. 121)。
川田拓也・林由華・岩崎勝一・大野剛 (2009)「琉球語宮古池間方言における mmya の談話機能」柴崎礼士郎 (編)『言語文化のクロスロード』111–130, 文進印刷
分析上の特徴
mmya が clitic として左側の要素に付くことが多い（伊良部も同様） → 「mmya のスコープが左に及ぶ」と漠然と想定している可能性が高い。実際、mmya の右側（後続の談話展開）への言及が非常に少ない。
川田他からの「宿題」
mmya と類似して見える naugara「なんか」、unu「その」との比較が必要。相互にどんな違いが？（川田他 2009: 128–129）
問題点：伊良部では例外が多すぎる
「背景となる情報の表示」では説明できない例が多数：
FOC（焦点）に付く例 — 焦点は「新情報」であり矛盾
obasan-ta=ga=du mmja sjeisjunzidai=tii…
「おばさんたちがFOC mmja 青春時代だと…」— =du は焦点標識。新情報を提示する要素に mmja が付く。

「そのまま捨ててSEQ mmja すぐ防空壕に…」— A 節も B 節も事態の連鎖。「背景化」とは言えない（cf. Hopper 1979）。

論点2 情報理論的性質

サプライザル S — ある語がどれだけ「驚き」か

S(w | ctx) = −log₂ P(w | ctx)
確率が低いほど S は高い（＝驚きが大きい）。bigram モデルでは ctx = 直前の1語。
具体例：ik（行く）の次に何が来るか？
ik の後に mmja が来る確率は約 19%（S = 2.4 bits = 驚きが小さい）。
maju（猫）が来る確率はほぼ 0（S = 11.8 bits = 驚きが大きい）。
3つの位置で S̄ を比較
mmja だけ、self が谷（3.72）→ post が急上昇（6.79）のV字型。unu / naugara は平坦。
注意：人間の予測は bigram のような狭い文脈窓に依存しない。ここでは「情報の流れ」という、このセクションで扱う問題を概念的に掴むために bigram を利用している。絶対値ではなく語間の相対的な差に意味がある。
bigram 的な mmja の性質
mmja は「直後の語を予測困難にする」ように見える。しかし、本当にそうか？
論点2 情報理論的性質

post S̄ が高い — 2つの仮説と検証

mmja の post S̄ は 6.79 と突出して高い。これは何を意味するか？ 2つの仮説が立てられる。

仮説 A：語彙仮説
mmja の後ろには珍しい語が集まる。つまり、mmja の後に理解困難なフェイズが始まる。
仮説 B：遷移仮説
後ろに来る語自体は普通だが、mmja が後続の品詞を制約しない（遷移分布が平坦）ため bigram 確率が低くなる。
仮説 A の予測
後続語の unigram S̄（文脈を無視した驚き度）がコーパス平均より高いはず。 → 低頻度語・特殊語が集中していれば文脈なしでも S が高くなる
仮説 B の予測
後続語の unigram S̄ は平均と変わらない。高いのは bigram S̄ だけ。 → 語自体は普通だが mmja→next の遷移が平坦なので bigram 確率だけが低い
post S̄ vs 頻度 C
頻度 C が大きいほど post S̄ は高い。mmja は回帰線のほぼ上（残差 +0.50）＝ 頻度だけでは説明しきれない「何か」がある。
実際の結果
mmja の後続語の unigram S̄ はコーパス平均と有意差なし（p = .060）。後ろに珍しい語が集まっているわけではない。
仮説 A：棄却
post S̄ の高さは、mmja が後続の品詞を制約しない（遷移分布が平坦）ことの数学的帰結。加えて頻度 C が大きいほど post S̄ は高くなる（R² = 0.42）。
✓ 仮説 B：支持

論点2 情報理論的性質

広い文脈窓で示唆されること（仮説）

聞き手は bigram ではなく n-gram 的な広い文脈で累積的に予測している。しかし、局所的に、間に何を挟むかで、X の予測力が Y に届くかどうかは変わりうる。

仮説：聞き手が広い文脈窓を活用するという前提に立てば、mmja は予測に情報を足さないので P(next|ctx, X, mmja) = P(next|ctx, X) が示唆される。ただし厳密な証明には trigram 以上の条件付き分析（＝大規模コーパス）が必要。
聞き手が bigram より広い文脈で予測しているという想定は妥当か？surprisal theory (Hale 2001, Levy 2008) では処理コストは文全体の文脈からの予測可能性に比例するとされ、読み時間研究 (Smith & Levy 2013) でも広い文脈の有効性が実証されている。
Hale, John. 2001. A probabilistic Earley parser as a psycholinguistic model. Proceedings of NAACL 2. 159–166.
Levy, Roger. 2008. Expectation-based syntactic comprehension. Cognition 106(3). 1126–1177.
Smith, Nathaniel J. & Roger Levy. 2013. The effect of word predictability on reading time is logarithmic. Cognition 128(3). 302–319.

問い：TGTが出たと知って、後続の予測はコーパス全体の分布からどれだけ変わるか？

なぜ両立するか — mmja の場合
gzip が VAFX→MMJA→N を圧縮パターンとして検出するのは事実。だが mmja は多様な境界タイプに出るため：

ISP→MMJA→V も多い
CP→MMJA→N も多い…

個別の n-gram はそれぞれ繰り返すが、mmja 後の品詞を全部合わせると、N も V も ISP も来うる → 条件付き分布が全体分布に近い → KL が低い。
では、KL が低い（＝後続の分布が変わらない）のに、なぜ mmja を消すと圧縮が悪化するのか？
gzip（LZ77）はバイト列の繰り返し一致長を測る。条件付き確率は見ていない。

論点3 mmja は一体なんなのか？

現時点で提起したい仮説

観察される事実
1
ナラティブに集中
mmja は1人語り（narrative）にばかり出現する。会話ではほとんど見られない。（近隣の池間方言について川田他 2009 も指摘）
2
談話構造の境界域に出現
情報構造の合間（TOP#COM の間、FOC#BCK の間）、等位接続の合間（「飛行機が来て#、戻って行ったら#、猫が食べ物を食べてしまっていたんだよ」的な連鎖節境界#）、話法交替のタイミング（「〜って#（あの人は）言ってね」）に多い。統語的に非均一な、しかしある種の「切れ目」に集中している。
3
「透明」な形態素
その出現によって、前後の文脈予測が大きく変わらない「透明」な形態素（全形態素で圧倒的に低い KL）。
仮説
mmja の挿入には、話者側の長い発話連続における発話計画が関与している。

情報構造助詞（TOP, FOC）や副動詞接辞（SEQ, COND, CSL）は、それぞれ個別の統語的境界を標示する。

mmja はそれらを超えた、より抽象的なレベルの「談話の切れ目」を示している。

個々の統語カテゴリに依存せず、「ここが区切りである」というメタ的な信号を挿入することで、ナラティブの談話構造を聞き手に伝達する。

いわば、口語における句読点的な機能を果たすのが mmja ではないか？
要約：mmja は統語カテゴリを超えた抽象的な談話境界シグナルであり、narrative において話者が発話の構造を聞き手に伝えるための手段ではないか。以下、この仮説を支持するデータを示す。

結論

mmja = 発話計画に基づくメタ談話標識

mmja vs 接続詞：何が同じで何が違うか
mmja aidu「すると」 assuga「しかし」
境界を表示 + + +
接続意味関係の明示 − + +
後続予測への影響 − (+) +
固定的な統語位置 − + +
mmja だけが、接続意味関係を明示せず、後続予測にも影響せず、固定的な統語位置も持たない、長い節連鎖構造の中でフレキシブルに談話境界を表示できる唯一の形態素。
n = 170 という突出した頻度 （cf. CONJ 全体で92: mata 36, aidu 24, assuga 5）
談話的に偏った意味（順接・逆接・転換など）を持たないからこそ、あらゆる境界で使える → スロットが多い → 高頻度。「句読点的」テクスト機能と整合的。
情報理論から見た mmja の位置づけ
デジタル通信では、データ列に同期マーカー（sync word）を挿入する。これは①ペイロード（情報内容）を持たず、②受信側にフレーム境界を知らせ、③周囲のデータに依存せず認識できる（Scholtz 1980）。mmja の性質 — KL が低く（①）、境界に出現し（②）、self S̄ が低い（③） — を理解する上で、Sync word のアナロジーは示唆的かもしれない。
Scholtz, R.A. 1980. Frame synchronization techniques. IEEE Transactions on Communications 28(8). 1204–1213.
結論：mmja は言い淀みではなく、発話計画に基づくメタ談話標識。接続意味関係を明示せず後続予測に影響しない「透明」な境界マーカーとして、ナラティブのテクスト構造を聞き手に伝達する — 口語における句読点。
